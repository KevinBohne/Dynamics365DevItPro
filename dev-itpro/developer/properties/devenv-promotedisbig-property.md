# PromotedIsBig Property
Sets a value that indicates whether the promoted action is displayed on the ribbon with a large icon.  
  
## Applies to  
  
- Page actions  
 
## Syntax
```
PromotedIsBig = true;
```
 
## Remarks  
Available options are **true**/**false**.
